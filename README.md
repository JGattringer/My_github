# 🚀 Git Commands Guide

Uma página web interativa e moderna que serve como guia de referência completo para comandos Git, desenvolvida para ajudar desenvolvedores no dia a dia.

## 📋 Sobre o Projeto

Esta aplicação web apresenta os principais comandos Git organizados por categorias, com explicações detalhadas e exemplos práticos. O projeto foi desenvolvido seguindo boas práticas de Git Flow e metodologias ágeis.

## ✨ Funcionalidades

- 📚 **32 comandos Git** organizados em 4 categorias principais
- 🎨 **Design moderno** com gradientes e efeitos glassmorphism
- 📱 **Totalmente responsivo** - funciona em mobile, tablet e desktop
- 🧭 **Navegação intuitiva** com menu sticky e scroll suave
- 💫 **Animações interativas** nos cards e elementos
- 🔍 **Interface limpa** focada na experiência do usuário
- ⚡ **Performance otimizada** com CSS puro

## 🗂️ Categorias de Comandos

### 🔰 Comandos Básicos

- Inicialização e configuração
- Status e histórico
- Staging e commits
- Visualização de diferenças

### 🌿 Branches

- Criação e navegação entre branches
- Merge e gerenciamento
- Comandos modernos (switch)

### 👥 Colaboração

- Repositórios remotos
- Push e pull
- Configurações de usuário
- Sincronização de equipe

### ⚡ Comandos Avançados

- Revert e reset
- Stash e rebase
- Cherry-pick
- Busca no histórico

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização moderna com:
  - Flexbox para layout
  - CSS Grid para responsividade
  - Animações e transições
  - Gradientes e efeitos visuais
  - Media queries para responsividade

```

```

## 🚀 Como Executar

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/seu-usuario/My_github.git
   ```

2. **Navegue até o diretório:**

   ```bash
   cd My_github
   ```

3. **Abra o arquivo no navegador:**
   - Abra `index.html` diretamente no seu navegador
   - Ou use um servidor local como Live Server (VS Code)
